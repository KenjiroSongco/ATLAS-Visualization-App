# ATLAS: Advancing Traffic Lights Algorithm System

## Project Overview
ATLAS (Advancing Traffic Lights Algorithm System) is an intelligent, algorithm-driven traffic light simulator designed to improve urban road management. Built to address the severe traffic congestion and road safety issues prevalent in major urban centers like Metro Manila, this system replaces outdated, static traffic timers with dynamic, real-time algorithms. 

By calculating a dynamic "Priority Score" based on vehicle queue lengths and average movement speeds, ATLAS efficiently distributes green light time to the lanes that need it most, minimizing idle time and maximizing overall traffic flow.

## Current Development Phase
This repository houses the Phase 1 Minimum Viable Product (MVP). It is a Java-based Command-Line Interface (CLI) application built using Object-Oriented Programming principles to validate the core mathematical logic, priority scaling, and safety constraints of the ATLAS algorithm.
